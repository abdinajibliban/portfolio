Burger Logger

Click the image above to see my burger logger that's deployed on heroku. For this homework assignment, I made a burger logger with MySQL, Node, Express, Handlebars and a homemade ORM. The user begins by writing the name of a burger. Then the user clicks the submit button. This will cause the name of the burger to appear. If the user then presses the "Devour It!" button then the name of the burger will be shown on the opposite side of the screen to indicate it has been devoured.
